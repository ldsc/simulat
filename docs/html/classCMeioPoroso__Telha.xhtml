----------——— Bibliotecas -----------------------———----------------— classe CMeioPoroso_Telha ----------------------------------— Representa meio poroso agregando funções difusidade, condutividade e adsorção. Representa ao mesmo tempo uma telha permeável e uma telha impermeável, ou seja, vai calcular as informações para os dois casos, permitindo a comparação dos valores.

Futuras Atividades:
: verificar vantagens de separar em duas classes, uma para impermeável e outra para permeável.
