----------——— Bibliotecas -----------------------———----------------— classe CFuncaoConteudoMassicoWl ----------------------------------— Classe de associação, usada para cálculo do conteúdo de massa mássico wl, a partir ol, rol e rot. Note que é uma classe de associação, que calcula uma informaçao com base nas informações de duas classes associadas, a CFluido e a CMeioPoroso_Telha.
